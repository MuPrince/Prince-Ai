# 前端项目 提示词

```
## role
前端开发工程师

## Tech Stack
- Vue 3 + TypeScript + Vite
- Element Plus（UI 组件库）
- Pinia（状态管理）
- Vue Router 4（路由管理）
- Axios + 请求拦截器（HTTP 封装）

## Module
client/web

## style
- 页面色彩以靛青和蓝色之间的颜色为主，保持一致的视觉风格

## Feature
项目骨架搭建、公共组件、请求封装、路由守卫、布局系统

## Requirements
1. **项目搭建**：Vite + Vue3 + TypeScript 初始化，配置路径别名 @/，统一代码风格 ESLint + Prettier
2. **请求封装**：基于 Axios 封装请求实例，统一 baseURL、超时时间、请求/响应拦截器
   - 请求拦截器自动注入 Authorization: Bearer <token>
   - 响应拦截器统一错误处理（401 跳转登录、500 提示服务异常）
3. **路由系统**：Vue Router 4 配置，支持路由守卫
   - / ：重定向到/home
   - /home：首页，展示功能列表，打开后默认展示RAG知识库功能标签页
4. **首页布局**：
    - 左右布局：左边功能列表，占用页面1/5，右边功能标签页，占用页面4/5
5. **功能标签页布局**：
    - 左右布局：标签页功能区域占用左边3/5，右边AI聊天窗口占用页面2/5，输入框高度为1/5，左右区域允许拖动调整大小
## 操作流程
1. 打开功能：点击功能列表中的功能，在右边打开对应的功能标签页。
2. 聊天窗口：用户可以在聊天窗口中输入消息，点击发送按钮，发送消息到服务器。
3. 消息记录：用户可以在消息记录中查看之前发送的消息记录。

## 功能列表（静态数据）
- RAG知识库（code：rag）
- 智能客服（code：service）

## 功能标签页
- RAG知识库（code：rag）
  - 标签页下方上传文件输入框
  - 上传的文件列表展示在上方4/5区域
- 智能客服（code：service）
  - 展示区域留白，仅构建聊天区域

## API 对接
| 前端操作 | 调用 API |
| --- | --- |
| 发送消息      | POST /code（功能列表中的code值）/chat |
| 上传文件      | PUT /rag/upload |
| 获取文件列表  | GET /rag/list   |



```
